# 🍊 Orange Chat Board

A playful chat UI designed to look like a glowing orange fruit — complete with a glossy highlight, rainbow stars in dark mode, and even a swaying leaf on top 🌿.  
It connects to a Flask backend with Ollama for real AI replies, and features a code panel that highlights generated code in ready-to-paste format.

---

## ✨ Features
- 🍊 **Orange-shaped chat board** with glossy highlight and swaying leaf
- 🌌 **Dark mode** with rainbow glowing drifting stars
- 🤖 **AI integration** via Flask + Ollama backend
- 🎨 **Code panel** with syntax highlighting (keywords, strings, comments)
- ⚡ Smooth animations and delightful UI effects
